The Challenge: Seeing in 3D

The Problem with 2D

A standard photograph is flat. It contains no information about the distance of objects. Our brains can *infer* depth from cues like shadows, perspective, and object size, but for a computer, it's just a grid of colored pixels.


The challenge is to create an algorithm that can look at a single 2D image and predict the real-world 3D distance for every single pixel.

The Solution: A Custom Deep Learning Model

An Encoder-Decoder Architecture

To solve this, we implemented a custom-built Convolutional Neural Network (CNN) based on the famous UNet architecture. This model is specifically designed for image-to-image tasks like depth estimation.


The model works in two parts: an Encoder (down-sampling path) that "sees" the image and captures its high-level features, and a Decoder (up-sampling path) that uses this information to reconstruct a detailed, pixel-perfect depth map.

Performance & Metrics

To validate the model, we evaluated it against the entire NYUv2 test set. The performance was measured using standard depth estimation metrics, where a lower score is better.

Output Abs Rel Error RMSE (meters)
Simon 0.1682 0.5693 m

These results are excellent and demonstrate the usage of this model. An average relative error of 16.8% and a typical error of only 57 cm is a brilliant result.

Real-World Applications
